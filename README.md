# REST API

The REST API to the example app is described below.

## Get list of Searchs

### Request

`GET /searchs/` -- > Get the list of everything searched in the API

    'http://localhost:8082/searchs

### Response

    []{"Urls" : "anything searched in the API since it was turned on"}
